# Open Lineage log files

The Open lineage logfiles are organized below by governance action. 

Each directory represents a governance action type running in the platform.  There is a file in this directory for each open lineage event.  Typically there is an event for the start of the action and one for the end.

Open Lineage events are described on the [main document website](https://egeria-project.org/features/lineage-management/overview/).
